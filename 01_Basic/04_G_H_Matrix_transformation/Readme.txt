H-Matrix.txt를 input으로 받아서  그에 해당하는 G-Matrix.txt를 생성하는 코드입니다.
H => G-Matrix 변환과정도 전부 출력해보는 것을 추천드립니다.

정답 확인 방법
1. input으로 드리는 H-Matrix.txt는 제공되어 있습니다. 해당 H-Matrix에 대해서 Answer 폴더에 있는 파일들과 같은 결과가 나오면 됩니다.

2. H-Matrix.txt를 input으로 받아서 그에 해당하는 systematic H-Matrix, 이에 해당하는 systematic G-Matrix, 기존 H-Matrix.txt에 대응하는 G-Matrix.txt를 각각 결과로 뽑아보시기 바랍니다.
 -> 즉, output으로 3개의 txt가 나오면 됩니다.
